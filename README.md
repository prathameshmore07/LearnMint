## 📋 Problem Statement

Design and develop a **fully responsive multi-page website** for a micro-course marketplace where instructors can sell short 30-minute skill-based modules. The platform allows students to explore courses, preview content, enroll, track progress, and unlock lessons dynamically.

**Industry:** EdTech

---

## 🗂️ Folder Structure

```
HTML & CSS PROJECT/
│
├── html/                  # All HTML pages (7 pages)
├── css/
│   └── style.css          # Global stylesheet
├── js/
│   ├── app.js             # Auth, navbar, dashboard, certificates
│   └── data.js            # Course data (instructors, lessons, videos)
├── Documentation/         # Case study and related docs
├── Screenshots
└── README.md
```

---

## 🚀 How to Run

1. Download or clone the project
2. Open `html/index.html` in any modern browser
3. No build tools or dependencies needed
